Who's in a shelter?

This is a graphical representation with the goal of highlighting trends in the demographic of people staying in shelters. Mainly encompassing children, adults, and whole families, this visualization will depict the total amount of each group in a shelter for a any given day from 08/21/2013 to 04/01/2024 along with their yearly averages.

Data Cleansing Process

  • A Data Set was found that could be downloadable as a CSV
  • Google Sheets was used to remove any unneccessary columns/rows and to add any other data
  • The Google Sheet was downloaded as a CSV
  • A python program was made to go through the data and turn it into a .json file
All data is from the DHS Daily Report from NYC Open Data
